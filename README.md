## NERDTree Commands

```
<ctrl> + t "Toggle nerdTree on/off
<ctrl> + f "Search 
```

### Pannel movement
```
<leader> + h Move the Selector Left 
<leader> + j Move the Selector Down
<leader> + k Move the Selector Up
<leader> + l Move the Selector Right  
```
